# Aero Soccer

GitHub Pages files:

- index.html: app shell
- styles.css: site styling
- app.js: app logic and fallback baked data
- data.csv: match/player stats loaded by the site
- combos.csv: goal scorer/assister timeline data
- settings.json: MVP weights, custom player stats, video links, and display settings

When publishing to GitHub Pages, keep these filenames exactly as written. The app fetches data.csv, combos.csv, and settings.json from the same folder as index.html.
